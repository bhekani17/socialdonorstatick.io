✅ Pre-Launch Checklist

  • Database created and schema imported
  • Environment file (.env) configured
  • Database credentials updated in php/config.php
  • Upload directories created and writable
  • PHP extensions installed (pdo_mysql, mbstring, openssl, fileinfo)
  • .htaccess file in place
  • Setup test passes all checks
  • Default admin password changed
  • Email settings configured (optional but recommended)
  • HTTPS enabled in production

🔌 API Endpoints

Authentication:
POST php/donor_login_mysql.php - Donor Login
POST php/donor_signup_mysql.php - Donor Registration
POST php/admin_login_mysql.php - Admin Login
GET php/logout.php - Logout

APIs:
GET php/dashboard_api.php - Dashboard Data
GET php/blood_requests_api.php - Blood Requests
